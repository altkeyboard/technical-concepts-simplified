# What is Zero Trust?

## Plain-English definition
Zero Trust is a security model that assumes no user or device should be trusted automatically—even inside the network.

## Core ideas
- Verify identity continuously
- Limit access to only what’s needed (least privilege)
- Monitor and log activity

## Why it matters
It reduces risk from stolen credentials, insider threats, and compromised devices.
